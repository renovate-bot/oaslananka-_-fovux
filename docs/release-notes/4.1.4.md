# Fovux 4.1.4 Release Notes

Fovux 4.1.4 republishes Fovux Studio with the current README screenshot URLs used by VS Code
Marketplace and Open VSX. It is a backward-compatible patch release.

## Headline Wins

- **Marketplace screenshots.** README image URLs point at the tracked
  `fovux-studio/resources/screenshots/` assets on `main`.
- **Version-aligned publish.** `fovux-mcp` and `fovux-studio` remain on the same monorepo release
  version.

## Breaking Changes

None.

## Release Validation

- `python scripts/check_versions.py`
- `pnpm package && pnpm check:bundle-size`
- `task ci`
- `task release:dry-run`
